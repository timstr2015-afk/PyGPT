## 🧠 Что такое PyGPT?

**PyGPT** — это полностью локальный искусственный интеллект для помощи в программировании на Python. В отличие от облачных сервисов (ChatGPT, Copilot), PyGPT работает **полностью на вашем компьютере** без доступа в интернет.

## ✨ Возможности

### 🎯 Генерация кода
- **Функции** — любые функции по описанию
- **Классы** — ООП с наследованием
- **Игры** — крестики-нолики, змейка, тетрис
- **Приложения** — калькулятор, блокнот, чат
- **API** — REST, GraphQL, Telegram боты
- **Алгоритмы** — сортировка, поиск, структуры данных

### 💬 Чат-помощник
- Объяснение концепций Python
- Помощь с ошибками
- Рекомендации по библиотекам
- Ответы на вопросы 24/7

### 🧠 Нейросеть
- **Transformer** архитектура (как у больших моделей)
- **Multi-head Attention** для понимания контекста
- **Обучение на лету** — становится умнее с каждым запросом
- **500+ примеров** в базе знаний

### 🌐 Сетевая синхронизация
- Обмен знаниями между ПК
- Автоматическое определение главного узла
- Сохранение всех данных локально
- Работа как майнер знаний

### 📚 Мега-обучение
- **Wikipedia** — статьи и документация
- **Google** — поиск примеров кода
- **Reddit** — вопросы и ответы
- **GitHub** — open-source проекты
- **YouTube** — туториалы
- **StackOverflow** — решения проблем
- **Русский язык** — словари, грамматика, правила
- **Английский язык** — словари, грамматика
